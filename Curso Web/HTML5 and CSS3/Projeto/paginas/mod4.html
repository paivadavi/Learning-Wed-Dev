Unidades de medida absoluta

O pixel do Css é um pouco maior do que o pixel do monitor

pt = points

pt: muito utilizado em fontes

Mais utilizados em sites que vão imprimir por possibilitar uma similaridade maior entre o que aparece na tela e a pós-impressão

Medida absoluta = não é influenciada por nada

Unidades de medida relativas (fonte)

em

O em é sempre baseado no font size do item pai

O em pega do item diretamente superiror a ele

O em vai aumentando a medida que você adentra as divs

rem

O rem se baseia sempre no elemento de maior hierarquia

Quando você não especifica nada, 1 re = 16 px

ex = utiliza a altura do x da fonte

ch

É o espaço de um caractere. Mas só funciona bem se estiver usando uma fonte monospace

Usamos o ch para delimitar o espaço entre cada caractere
