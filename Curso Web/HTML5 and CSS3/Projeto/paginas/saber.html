Cores no CSS

O ideal é não usar as cores pelo nome. Pois o HTML e o CSS tem uma base de dados limitada para cores com palavras.

É muito importante aprender o esquema RGB

Usando os esquemas RGB e Hexadecimal, não há risco das cores

Esquema RGB

Utiliza as gradações das cored Red, green e Blue para chegar à cor desejada

Utiliza um esquema numerico para a representação das cores

R: 0 G: 0 B: 0 Cor: Preto

R: 225 G: 225 B: 225 Cor: Branco

R: 225 G: 0 B: 0 Cor: Vermelho

R: 0 G: 225 B: 0 Cor: Verde

R: 0 G: 0 B: 225 Cor: Azul

Esquema Hexadecimal

Também utiliza as gradações das cores 3 cores primárias para chegar à cor desejada. Mas faz isso através da repesentação Hexadecimal
